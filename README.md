# makeyourburger

Projeto de pedidos online para hamburgueria desenvolvido em Vue JS 3 com backend em um JSON Server.

## Project setup
```
npm install
```

### Compila o código em modo desenvolvimento
```
npm run serve
```

### Ativa o backend em JS
```
npm run backend
```
